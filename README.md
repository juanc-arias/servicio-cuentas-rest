# Servicio REST de Consulta de Cuenta

Este proyecto implementa un servicio REST básico para consultar la información de una cuenta bancaria de forma simulada (mockeada), usando Python y FastAPI.

## Objetivo

Este servicio se construyó como ejemplo de una funcionalidad que podría exponerse desde un sistema bancario legado mediante una arquitectura orientada a servicios, con el fin de mostrar una interfaz clara, desacoplada y fácil de consumir.

## Tecnologías utilizadas

- Python 3.13
- FastAPI
- Uvicorn

## Requisitos

Antes de ejecutar el proyecto, se debe tener instalado:

- Python 3.13 o superior
- Visual Studio Code (opcional)
- PowerShell o terminal de comandos
